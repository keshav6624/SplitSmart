# 💸 SplitSmart-- Real-time Group Expense Tracker

A simple and intuitive **Splitwise-like expense management app** built using **Streamlit**.
It helps users track shared expenses, manage balances, and understand who owes whom in a group.

---

## 🔍 Overview

Managing shared expenses with friends, roommates, or teams can get messy.
**ExpenseSync** solves this by providing a clean interface to:

* Add and track expenses
* Split costs among users
* View balances and contributions

---

## ✨ Features

* ➕ Add expenses with title and amount
* 👥 Track multiple users in a group
* 💰 View individual balances
* 📊 Manage shared expenses easily
* 📂 Organized multi-page Streamlit interface

---

## 🧠 Tech Stack

* **Frontend & App Framework:** Streamlit
* **Backend Logic:** Python
* **Database:** SQLite
* **Version Control:** Git + GitHub

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/keshav6624/SPLITWISE.git
cd SPLITWISE
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
python -m streamlit run app.py
```
